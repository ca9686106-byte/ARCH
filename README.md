# Multi-Agent AI System Architecture

This repository contains a multi-agent AI system architecture with four specialized AI agents:

1. **Coder**: Responsible for generating and debugging code.
2. **Designer**: Manages user interface and user experience design.
3. **Bug Finder**: Identifies bugs and suggests fixes within the code.
4. **Coordinator**: Oversees the project flow and agent interactions.

## Getting Started

To set up and run this multi-agent AI system, follow the instructions in the README.md file and configure the system using the config.py file.

## Structure
- `requirements.txt`: Lists necessary dependencies.
- `config.py`: Configuration settings for the agents.
- `coder_agent.py`: Implementation of the Coder agent.
- `designer_agent.py`: Implementation of the Designer agent.
- `bug_finder_agent.py`: Implementation of the Bug Finder agent.
- `coordinator_agent.py`: Implementation of the Coordinator agent.
- `main.py`: Main entry point to run the system.
- `utils.py`: Shared utilities for the agents.

## Usage

Run `main.py` to start the multi-agent AI system.
